Machine Learning in Analyzing Cell Mechanical Behavior

A number of machine learning techniques are applied in the characterization and analysis of multiple cancer cell lineage morphologies. Deep convolutional neural nets are utilized to perform instance segmentation on brightfield cell images, allowing for extremely high-throughput label-free data collection. From the segmentation output, single cell morphological properties and cell network properties can be measured. Fractal dimension is calculated from the binarized segmentation output, and graph properties are calculated from a node&edge representation of the cell network. Hierarchical clustering is performed on characterized cell populations in order to identify phenotypically distinct lineages. Complete code can be viewed on github: [CNN segmentation] [morphological analysis].
