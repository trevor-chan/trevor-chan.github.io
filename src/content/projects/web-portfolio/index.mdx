## Overview

Design and implementation of this portfolio website. HTML, CSS, and select JavaScript written from scratch.

## Approach

Page transitions use the JS Smoothstate plugin by Miguel Perez. The layout draws on influences from minimal editorial design — fixed-width columns, sparse typography, and restrained use of color. Complete source code is available on GitHub.
